# Online-Fasion-Store

## Technical Stack

- Front-end: React.js
- Back-end: Spring, Hibernate, MySQL
- API: RESTful API

## Entities

- Customer
- Product
- Shopping Cart
- Order

Product:
- product ID
- category ID
- name
- price
- image link
- post date
- color
- score
- description

Product Categories:
- category ID
- category name
- parent category id (nullable)
